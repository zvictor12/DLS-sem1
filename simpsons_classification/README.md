# Simpsons Classification

## О задаче

Классификация персонажей сериала "Симпсоны" по изображениям. Требуется предсказать класс персонажа для тестовых картинок и подготовить сабмит.

## Что сделано

- загружен и подготовлен датасет изображений;
- реализован кастомный `Dataset` для работы с файлами;
- добавены аугментации и оверсемплинг редких классов;
- обучены сверточные модели на PyTorch;
- подготовлены предсказания для соревнования.

## Артефакты

- `Классификация симпсонов.ipynb` — основной ноутбук;
- `densenet_model.pth` — сохраненные веса модели.

- Python
- PyTorch
- torchvision

- Pillow
