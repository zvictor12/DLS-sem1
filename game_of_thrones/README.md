# Game of Thrones Survival Prediction

## О задаче

В ноутбуке решается задача бинарной классификации персонажей из вселенной Game of Thrones: по биографическим признакам нужно предсказать, останется ли персонаж жив.

## Что сделано

- разобраны признаки датасета `A Wiki of Ice and Fire`;
- обработаны пропуски и категориальные признаки;
- проведен EDA целевой переменной и влияющих факторов;
- подготовлены данные для обучения;
- обучены и сравнены классические модели классификации.

## Стек

- Python
- pandas, numpy
- scikit-learn
- matplotlib, seaborn
