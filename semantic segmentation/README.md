# Semantic Segmentation

## О задаче

В ноутбуке решается задача бинарной сегментации медицинских снимков из датасета PH2/ADDI: по изображению нужно предсказать маску области интереса. Качество моделей оценивается по метрике IoU.

## Что сделано

- подготовлены датасет, аугментации и `DataLoader`;
- реализована метрика IoU для бинарной сегментации;
- реализованы и проверены BCE, Dice Loss и Focal Loss;
- реализованы архитектуры SegNet и U-Net на PyTorch;
- обучены модели с разными функциями потерь;
- добавлены визуализации предсказанных масок и сравнение результатов.

## Результаты

Лучший результат на валидации показала модель `UNet + BCE`:

| Model | Best validation IoU | Final validation IoU |
| --- | ---: | ---: |
| UNet BCE | 0.8251 | 0.8251 |
| UNet Focal | 0.8184 | 0.7844 |
| UNet Dice | 0.8050 | 0.7474 |
| SegNet Focal | 0.7641 | 0.7093 |
| SegNet Dice | 0.7636 | 0.6646 |
| SegNet BCE | 0.7559 | 0.7559 |

По итогам экспериментов U-Net оказался сильнее SegNet по IoU и обучался быстрее в рамках проведенного сравнения.
