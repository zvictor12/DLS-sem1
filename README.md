# Deep Learning School — Semester 1

Репозиторий с домашними заданиями первого семестра Deep Learning School. Каждая работа вынесена в отдельную папку с ноутбуком и кратким описанием.

## Содержание

| Папка | Тема |
| --- | --- |
| [`game_of_thrones`](game_of_thrones/) | Классическая ML-классификация выживания персонажей Game of Thrones |
| [`user_outflow`](user_outflow/) | Kaggle-задача прогнозирования оттока пользователей |
| [`fully_connected_and_cnn`](fully_connected_and_cnn/) | Полносвязные и сверточные нейронные сети на PyTorch |
| [`simpsons_classification`](simpsons_classification/) | Классификация персонажей "Симпсонов" по изображениям |
| [`semantic segmentation`](semantic%20segmentation/) | Бинарная сегментация медицинских снимков |
| [`object_detection`](object_detection/) | Улучшение детектора объектов на PyTorch |

## Стек

- Python
- PyTorch
- scikit-learn
- pandas, numpy
- matplotlib, seaborn
