# User Outflow Prediction

## О задаче

Решение соревнования по прогнозированию оттока пользователей. Нужно предсказать вероятность положительного класса и оптимизировать качество по ROC-AUC.

## Что сделано

- проведен первичный анализ данных и пропусков;
- изучены распределения числовых и категориальных признаков;
- подготовлены пайплайны для линейных моделей;
- протестированы Logistic Regression и градиентный бустинг;
- подготовлен файл с предсказаниями для отправки в контест.

## Стек

- Python
- pandas, numpy
- scikit-learn
- CatBoost
- matplotlib, seaborn
